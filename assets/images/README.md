## Image Guidelines

### Recommended Sizes
- **Featured Images**: 1200x630px (for social media sharing)
- **Article Headers**: 1200x600px
- **Thumbnails**: 400x300px

### Format
- Use JPEG for photographs
- Use PNG for graphics with transparency
- Optimize images for web (compress to reduce file size)

### Naming Convention
- Use lowercase
- Use hyphens instead of spaces
- Be descriptive: `gilas-basketball-celebration.jpg`

## Adding New Images

1. Place image files in this directory
2. Reference them in your markdown posts using:
   ```
   image: "/assets/images/your-image-name.jpg"
   ```
